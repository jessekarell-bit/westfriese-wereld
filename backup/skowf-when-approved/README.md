# Backup: Partnerspagina met SKOWF-sectie

Deze map bevat de **partnerspagina zoals die was toen SKOWF nog als strategisch partner op de site stond**. De live partnerspagina is neutraal gemaakt (geen SKOWF-sectie) totdat de samenwerking is bevestigd.

## Terugzetten wanneer SKOWF goedkeuring geeft

1. Kopieer het bestand **`app/partners/page.tsx`** uit deze backup-map naar **`app/partners/page.tsx`** in de projectroot (overschrijf het huidige bestand).
2. De component `components/SKOWFStrategicPartner.tsx` staat nog in het project en wordt dan weer op de partnerspagina getoond.

Er hoeft verder niets te worden hersteld: de SKOWF-component is niet verwijderd, alleen niet meer op de partnerspagina in gebruik.
